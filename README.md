# Python Learning Notebook Collection

This repository contains a set of Python Jupyter notebooks designed to demonstrate and practice core Python programming concepts, data analysis, and visualization.

## Contents

- `Control_flow,_collection_data_types.ipynb`
  - Control flow statements (`if`, `for`, `while`)
  - Data types and collections (`list`, `tuple`, `set`, `dict`)
  - Basic operations and examples

- `Functions,_OOPS.ipynb`
  - Python function definitions, parameters, and return values
  - Object-oriented programming fundamentals
  - Classes, objects, methods, and inheritance examples

- `More_Python_program.ipynb`
  - Additional Python examples and practice programs
  - Problem-solving with Python constructs

- `Numpy_examples.ipynb`
  - NumPy array creation and manipulation
  - Numerical operations and vectorized computation

- `Pandas_DeepDive.ipynb`
  - Data loading and manipulation with pandas
  - DataFrame operations, filtering, and analysis

- `Seaborn_examples.ipynb`
  - Data visualization using seaborn
  - Plot types, styling, and exploratory data analysis

## Getting Started

1. Install Python and Jupyter Notebook / JupyterLab.
2. Install the required libraries if not already available:

```bash
pip install notebook numpy pandas seaborn matplotlib
```

3. Open the notebooks from the `Python/` folder with Jupyter Notebook or JupyterLab.

```bash
jupyter notebook Python/
```

## Purpose

This project is useful for learning and reviewing Python programming basics, working with the main data science libraries, and exploring example workflows in Jupyter notebooks.

## Notes

- Each notebook is self-contained and focuses on a specific topic.
- Use the notebooks interactively to run examples and modify code.
